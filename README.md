# 🐦 Flappy Bird Clone

A simple Flappy Bird-inspired game developed using Unity and C#. The player controls a bird and navigates through obstacles while trying to achieve the highest score possible.

## 🎮 Features

* Smooth bird movement
* Obstacle spawning system
* Score tracking
* Collision detection
* Game Over system
* Simple and clean UI

## 🛠️ Built With

* Unity
* C#
* Unity Physics
* Unity UI

## 🎯 Gameplay

* Press Space to make the bird flap.
* Avoid hitting pipes.
* Pass through obstacles to earn points.
* Try to achieve the highest score.

## 🎮 Controls

| Key   | Action                   |
| ----- | ------------------------ |
| Space | Flap                     |
| R     | Restart (if implemented) |

## 🚀 Getting Started

1. Clone the repository.
2. Open the project in Unity Hub.
3. Open the main scene.
4. Press Play.
